# CocoRoot

Este repositório contém o projeto **CocoRoot**, desenvolvido no âmbito da Unidade Curricular de **Projeto** da Universidade Europeia / IADE.

Este projeto consiste no desenvolvimento de uma aplicação web voltada para a **agricultura inteligente, com foco no uso do substrato de fibra de coco.** A plataforma permite que pequenos agricultores gestionem os seus cultivos de forma mais eficiente, combinando técnicas modernas com ferramentas digitais acessíveis. Através de dashboards intuitivos, os utilizadores podem monitorizar informações essenciais sobre as suas produções. Além disso, a aplicação promove a troca de conhecimentos entre agricultores, incentivando a partilha de experiências e a melhoria contínua das práticas agrícolas. O grande objetivo é capacitar os pequenos produtores, ajudando-os a aperfeiçoar as suas técnicas e a obter melhores resultados nas suas colheitas.
